## CHUCKN API
php 8.0 & Symfony 5.4 project
___
WebSite & API (in json format) to give Chuck Norris Facts in French.

- search engine
- random facts
- rating system
___
### API use:

Get a random Fact:

> http(s)://x.x.x.x/api/rand

Get a Facts from an id:

> http(s)://x.x.x.x/api/joke/{id}
